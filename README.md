# SimuladorFinanciero-Backend

Backend - Proyecto de la clase Teoría de la Simulación.

</br>

## Archivo .env

<table>
  </tr>
    <td>
      <code>URL_DB</code>
    </td>
    <td>
      Enlace de la Base de Datos de MondoDB.
    </td>
  <tr>
  <tr>
    <td>
      <code>PORT</code>
    </td>
    <td>
    Puerto del Backend.
    </td>
  </tr>
  <tr>
    <td>
      <code>PORT_FRONTEND</code>
    </td>
    <td>
    Puerto del Frontend.
    </td>
  </tr>
  <tr>
    <td>
      <code>TOKEN_SECRET</code>
    </td>
    <td>
    String para cifrar y descifrar el token.
    </td>
  </tr>
</table>
</br>

## Comandos

<table>
  </tr>
    <td>
      npm install
    </td>
    <td>
      Instalar dependencias de Node.
    </td>
  <tr>
  <tr>
    <td>
    npm run dev
    </td>
    <td>
    Inicia el servidor.
    </td>
  </tr>
</table>
